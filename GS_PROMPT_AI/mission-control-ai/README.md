# Mission Control AI — AgroSat

## Integrantes

* Nome 1 — RM 571709


## Trilha Escolhida

🌾 AgroSat — Sensoriamento Agrícola

## Objetivo

Desenvolver um sistema inteligente de monitoramento operacional para um satélite multiespectral semelhante ao CBERS-4A, utilizando IA generativa para interpretar dados de telemetria e traduzir impactos técnicos para o agronegócio brasileiro.

## Telemetrias Monitoradas

* Saúde do sensor NDVI
* Temperatura do payload óptico
* Capacidade de armazenamento
* Janela de downlink
* Estabilidade de atitude
* Energia disponível

## Impacto na Sociedade

O AgroSat auxilia produtores rurais, agrônomos e seguradoras agrícolas através do monitoramento orbital de lavouras, permitindo decisões mais precisas sobre irrigação, produtividade e riscos agrícolas.
